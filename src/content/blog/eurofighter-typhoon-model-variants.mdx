The **Eurofighter Typhoon model** and capability standard is defined by "Tranches." Each new Tranche brings significant software and hardware upgrades.

| Tranche Model | Primary Role | Key Upgrades |
| :--- | :--- | :--- |
| **Tranche 1** | Air Superiority | Basic multirole, legacy avionics. |
| **Tranche 2** | Multirole | Improved computing, guided bombs. |
| **Tranche 3** | Swing-role | E-scan radar provisions, conformal fuel tanks. |
| **Tranche 4+** | Advanced Multirole | Fully integrated AESA radar, modern weapons. |
